Quick Split

The developer tool for fast, simple A/B tests

Get started

Download the CLI (requires Python3.7+ and pip installed on your system).

$ pip install --user quicksplit

Register a new account.

$ quicksplit register

Create an experiment. You'll use the name of the experiment in the next steps, where you'll log exposure and conversion events to your experiment.

$ quicksplit create my-experiment

Locate the public, staging token for your account. Staging tokens are used to separate staging data from production data.

$ export TOKEN=$(quicksplit tokens --staging --public)

Create an exposure for your experiment by sending JSON encoded data to the service.

$ curl https://api.quicksplit.io/exposures \
  -H "Authorization: ${TOKEN}" \
  -d '{
    "experiment": "my-experiment",
    "subject": 12345,
    "cohort": "control"
   }'

Exposures have three required fields: experiment, subject, and cohort.

  • The experiment is the name of the experiment you created previously.
  • The subject is a unique identifier, typically as `user_id` provided by your own backend.
  • The cohort corresponds to which version of your application the user is experiencing. In an A/B test, you might use the labels "A" and "B" for your cohorts.

Confirm that the logging was successful by checking the recent events.

$ quicksplit recent

┌────────────┬─────────────────┬─────────┬─────────┬───────┬───────────────────────────────┐
│ Type       │ Experiment      │ Subject │ Cohort  │ Value │ Last seen                     │
├────────────┼─────────────────┼─────────┼─────────┼───────┼───────────────────────────────┤
│ exposure   │ my-experiment   │ 12345   │ control │       │ Sun, 23 Feb 2020 23:14:02 GMT │
└────────────┴─────────────────┴─────────┴─────────┴───────┴───────────────────────────────┘

Create a conversion for your experiment.

$ curl https://api.quicksplit.io/conversions \
-H "Authorization: ${TOKEN}" \
-d '{
"experiment": "my-experiment",
"subject": 12345
}'

Download the results of your experiment.

$  quicksplit results my-experiment

┌─────────┬──────────┬─────────────────┬────────────────────┐
│ Cohort  │ Subjects │ Conversion rate │ 95% conf. interval │
├─────────┼──────────┼─────────────────┼────────────────────┤
│ control │ 1        │                 │ [,]                │
└─────────┴──────────┴─────────────────┴────────────────────┘
