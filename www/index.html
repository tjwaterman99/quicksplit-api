Quick Split

The developer tool for fast, simple A/B tests

Get started

Creating a split test requires registering an account with quicksplit.io

You can download the CLI to create your account.

curl -o /usr/local/bin/quicksplit https://quicksplit.io/cli
quicksplit register

You'll be prompted for an email address and password. We ask for an email address so that you can reset your password in case you ever lose it.

Now that your account is created, you can start creating A/B tests. We'll describe how to create a test that measures the affect of changing your product's landing page on its signup rate.

First, we'll need to create an Experiment that you'll use to associate each of your products users with.

export NAME="My First Experiment"
quicksplit create --name $NAME
